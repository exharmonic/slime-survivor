## Gameplay Features
* **Endless Survival:** Battle against increasingly difficult waves of slime enemies.
* **Boss Fights:** Test your skills and dodge patterns against the giant Slime Boss.
* **Dynamic Combat:** Utilize different weapons and abilities (including pistols and boomerangs) to clear the arena.
* **Custom UI:** Track your health and progress with an integrated on-screen interface.
* **Compass:** I wonder where the compass leads one to?

## Controls
| Action | Key / Input |
| :--- | :--- |
| **Move** | `W` `A` `S` `D` or `Arrow Keys` |
| **Aim** | `Mouse Cursor` |
| **Attack / Shoot** | `Left Mouse Click` |

## Running the Project
1. Clone this repository to your local machine: 
   ```bash
   git clone https://github.com/exharmonic/slime-survivor.git
   ```
2. Open the **Godot 4.5 Project Manager**.
3. Click **Import** and navigate to the cloned folder to select the `project.godot` file.
4. Hit **F5** (or the Play button in the top right) to run the game!
